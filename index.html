Andrew's Guide to Linux Distros

What is a Linux Distro?

This website provides a guide to help with the important task of choosing the best Linux Distro for you. You may be wondering, what exactly is a Linux distro?

At its most basic level, a linux distro is a complete operating system built around the Linux system. In addition to the kernel, they will typically include(at a minimum) the GNU Core Utils, a compiler, and a shell.
